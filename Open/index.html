Le festival EXIT, également connu sous le nom de State of EXIT, est un festival de musique qui a lieu tous les étés dans la forteresse de Petrovaradin à Novi Sad, en Serbie. Le festival EXIT se tient dans la forteresse de Petrovaradin, qui date du XVIIIᵉ siècle et domine le Danube.
